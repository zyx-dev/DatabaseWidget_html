User Guide
1. Enter your Eid (lower case) and click \'Confirm\'.
2. If you type incorrectly, click \'No\' in the popup window.
If right, click \'Yes\'.
Compare Two Products
①Open the files you want to compare.
Please select one file first, open it and then select another. Also you can select several files first, then click \'Open Second Files\' to select another several files.
②Click \'Two Products\' and it will be generated one or more HTML.
③You can find the result folder named \'db\' on your desktop.
Extract Band Information
①Open the files you want to compare.
You can select one or more files at a time.
②Click \'PV_FV\' and it will be generated one or more files with only band information.
③You can find the result folder named \'db\' on your desktop.
Download Database
①Click \'Download Database\', fill username, password and product in the popup window.
②Click \'Confirm\'. If right, click \'Yes\'. You can minimize all Windows.
③You can find the result folder named \'db\' on your desktop.
